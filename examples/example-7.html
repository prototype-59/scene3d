3D Cuboid nodes graph

dataset: Les Miserables: coappearance weighted network of characters in the novel Les Miserables. D. E. Knuth, The Stanford GraphBase: A Platform for Combinatorial Computing, Addison-Wesley, Reading, MA (1993).

$(function() { var myCanvas = $('#3dcanvas').scene3d({ width:"500", height:"500", background: "#eeffff", fontFamily: 'Arial', fontSize: 0.4, ViewpointPosition: '60 50 170', ViewpointOrientation: '-0.5 0.5 0 0.5', }); $.getJSON( "miserables.json", function( myGraph ) { myCanvas.addGraph({ type: 'cuboid', radius: 2, data: myGraph, width: 100, height: 100, depth: 100, fontSize:2, labeloffset:2 }); }); });
